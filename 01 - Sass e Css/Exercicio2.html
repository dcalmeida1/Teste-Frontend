Exercício 02 - Sass e CSS

Seletor "+"

O sinal “+” é utilizado quando se deseja selecionar o primeiro elemento que esteja localizado imediatamente após um determinado elemento

No exemplo abaixo, somente o primeiro parágrafo que estiver localizado logo após a tag <a> (classe link1) será modificado.

Uso do seletor "+"

a.link1 + p {

    color: red;

}

Código em html

            <div>
              <a class="link1"
                   "href="http://www.google.com.br"
                    target="_blank">
                    Link externo
              </a>
              <p>Texto modificado pelo seletor</p>
              <p>Texto não modificado pelo seletor</p>
            </div>

Resultado

Link externo

Texto modificado pelo seletor

Texto não modificado pelo seletor

Seletor "~"

O sinal “~” é utilizado quando se deseja selecionar todos os elementos que são precedidos por um determinado elemento.

No exemplo abaixo, todos os elementos que forem precedidos pela tag <a> (classe link2) serão modificados.

Uso do seletor "~"

a.link2 ~ p {

    color: blue;

}

Código em html

            <div>
              <a class="link2"
                   "href="http://www.google.com.br"
                    target="_blank">
                    Link externo
              </a>
              <p>Texto modificado pelo seletor</p>
              <p>Texto modificado pelo seletor</p>
              <div>
                <p>Texto não modificado pelo seletor</p>
                <p>Texto não modificado pelo seletor</p>
              </div>
            </div>

Resultado

Link externo

Texto modificado pelo seletor

Texto modificado pelo seletor

Texto não modificado pelo seletor

Texto não modificado pelo seletor

Observação:

Para ambos os casos, os elementos precisam pertencer ao mesmo elemento pai para o seletor surtir efeito.

Diferenças

A principal diferença que pode ser constatada é que o seletor "+" busca pelo primeiro elemento adjacente ao elemento de referência, enquanto o seletor "~" busca por todos os elementos precedidos pelo elemento de referência.
