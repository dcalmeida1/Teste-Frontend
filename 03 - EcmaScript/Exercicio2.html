Exercício 02 - EcmaScript

vetor.forEach(...) x vetor.map(...)

vetor.forEach(function(a, b){ vetor[b] += 2; })
vetor.map( i => i += 2)

A principal diferença entre os trechos de códigos acima é que o forEach faz a iteração sob cada elemento do array e executa a operação, contida no corpo da função, em cada elemento do array, e retorna o mesmo array modificado. Por sua vez, o map executa também a iteração sob cada elemento, executando a operação contida no corpo da função, mas retornando um novo array como resultado.

Essa diferença é importante, pois no primeiro caso o vetor original é substituído e no segundo caso o vetor original é preservado.
