Exercício 03 - EcmaScript

[   ].push(..)] x [   ].concat(...)

[   ].push(2)]
[   ].concat([2])

O [].push(2) é um método que adiciona o número 2 dentro do array original ([ ]). O [].concat([2]) é um método que adiciona o número 2 em uma cópia do array original retornando um novo array como resposta.

Essa diferença é importante, pois no primeiro caso o array original é substituído por outro array, e no segundo caso é criado um novo array e o array original é preservado.
